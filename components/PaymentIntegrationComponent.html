Description

Payment element component handles Stripe payment processing for cart checkout. This component provides a standalone payment form that handles payment intent creation, Stripe Elements rendering, and payment confirmation.

Prerequisites

Include the Stripe.js library in your index.html before using this component: ```html ```

Preview

Usage

import { PaymentIntegrationModule } from '@congarevenuecloud/elements';

@NgModule({
imports: [PaymentIntegrationModule, ...]
})
export class AppModule {}

<apt-payment-integration
